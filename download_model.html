🤖 ML Model Setup for Quiz Generation

Download the AI model to enable intelligent quiz generation

📦 Model Details:
  • Name: all-MiniLM-L6-v2 (quantized)
  • Size: ~8MB
  • Type: Sentence Transformer (BERT-based)
  • Purpose: Semantic understanding for better quiz distractors
1 Download the Model File

Click the button below to download the ML model from HuggingFace:

⬇️ Download Model (8MB)
⚠️ Important: The file will download as all-MiniLM-L6-v2-quant.tflite. You need to rename it to sentence_encoder.tflite in the next step.
2 Place the Model in the Correct Location

Move the downloaded file to:

Eduthon-flutter\assets\models\sentence_encoder.tflite

Full path:

c:\Users\mohan\Desktop\IIT_Bombay(Hack)\Eduthon-flutter\assets\models\sentence_encoder.tflite
3 Verify the Installation

Open PowerShell in the project directory and run:

cd c:\Users\mohan\Desktop\IIT_Bombay(Hack)\Eduthon-flutter dir assets\models\sentence_encoder.tflite

You should see a file of approximately 8-9 MB.

4 Build and Test

Run the Flutter app:

flutter pub get flutter run

Or build the APK:

flutter build apk --release
✅ When it's working, you'll see these logs:
🔄 [EnhancedQuizGen] Initializing ML service... ✅ [MLQuizService] ML model initialized successfully 🤖 [EnhancedQuizGen] Using ML to rank distractors ✅ [EnhancedQuizGen] Got 3 ML-ranked distractors
